What is Drizzle CSS?

A classless-first CSS framework driven by CSS variables, bundled with Lightning CSS.

Why classless?

Because semantic HTML deserves to look good without ceremony. Classes enhance, they don't gate.

Prose

.prose constrains content to --content-width and centres it — useful for long-form text inside a wider .container wrapper. On this page the body is already at content-width, but the pattern is: 

<div class="container">
  <div class="prose">
    <h1>Article title</h1>
    <p>Long-form content, centred within a wide layout.</p>
  </div>
</div>
