Drizzle CSS

A classless-first CSS framework. Bare semantic HTML renders well — classes only when you need more.

Source

What it is

Drizzle styles plain semantic HTML using CSS variables. No build step needed for consumers — drop one <link> tag and write markup. Customise by overriding any --variable.

This very page uses no classes for layout — only for the theme toggle button variant and the lead paragraph.

Quick example

Bare markup:

<article>
  <h1>Hello</h1>
  <p>World</p>
  <button>Click</button>
</article>

Theming

Every visual is driven by CSS variables on :root. Override one to retheme. Example — change the link colour to red: 

:root { --color-link: #ff0000; }

Dark mode

Auto-respects prefers-color-scheme. Add .dark or .light to <html> to force a mode. The toggle above demonstrates the class override.
